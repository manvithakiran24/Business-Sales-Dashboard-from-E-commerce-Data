# Business-Sales-Dashboard-from-E-commerce-Data
Power BI dashboard for e-commerce sales analysis. Visualizes sales trends, product categories, customer segments (Consumer/Corporate/Home Office), and geographic performance. Interactive slicers enable filtering by date, category, and region. Provides actionable insights for inventory, marketing, and regional expansion strategies.

## Overview
This dashboard visualizes sales data from 2011-2014 to help business owners make data-driven decisions about inventory, marketing, and regional strategies.

## Features
- **Sales Trends**: Line chart tracking revenue over time
- **Category Analysis**: Bar chart comparing product category performance
- **Customer Segments**: Pie chart showing Consumer, Corporate, and Home Office sales distribution
- **Geographic View**: Interactive map displaying sales by location
- **Key Metrics**: Cards showing total sales, orders, and profit
- **Interactive Filters**: Slicers for date, category, and region

## Tools Used
- Power BI Desktop
- CSV data (Superstore dataset)

##  Key Insights
- Identified best-selling product categories
- Discovered seasonal sales patterns
- Analyzed customer segment performance
- Mapped high-value geographic markets
